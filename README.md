# Tumblr Statistics

1. create an app on http://www.tumblr.com/oauth/apps
2. click explore api and fetch the 4 variables below

```
TUMBLR_CONSUMER_KEY
TUMBLR_CONSUMER_SECRET
TUMBLR_OAUTH_TOKEN
TUMBLR_OAUTH_TOKEN_SECRET
```
